MySQL 엔진은 요청된 SQL 문장을 분석하거나 최적화하는 등 DBMS의 두뇌에 해당하는 처리를 수행합니다.

MySQL엔진을 구성하는 요소는 다음과 같습니다.
- 커넥션 핸들러
- SQL 파서
- 전처리기
- 옵티마이저

MySQL은 표준 SQL 문법을 지원하기 때문에 표준 문법에 따라 작성된 쿼리는 타 DBMS와 호환되어 실행될 수 있습니다.

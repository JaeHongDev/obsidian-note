복사된 데이터가 준비됐지만
- 복사된 데이터는 소스 서버에서 백업을 실행한 과거 데이터
- 백업 시점 이후에 새롭게 변경된 데이터는 레플리카 서버에 적용도 있지 않음
- 실시간으로 변경되는 데이터도 레플리카로 서버로 복제되지 않음

아래 명령어를 통해 소스서버와 레플리카 서버간의 동기화가 진행된다.
